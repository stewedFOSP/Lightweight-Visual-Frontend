A Lightweight Dual-Branch Visual Front-End for Efficient Audio-Visual Target Speaker Extraction

Qualitative demos for the dual-branch visual front-end. Attention maps are shown first; audio comparisons below use models trained with tolerance = 5 and early_stop = 15 (IIANet backbone).

Compared visual front-ends: ResNet18 (pretrained), BlazeNet64, and the proposed method (32×32 + lmk84).
